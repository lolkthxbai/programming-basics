# programming-basics

Aquí intento explicar algunos conceptos básicos comunes de los lenguajes de programación modernos para ayudarte a entender mejor qué significa «programar» o escribir «code».

## ¿Qué es 'code'?

Lo que llamamos «código» son las palabras escritas que utilizamos para comunicarnos con las computadoras. Por lo general, son instrucciones que la computadora debe ejecutar o «run». Cuando escribimos código con una estructura organizada que se puede ejecutar, lo llamamos un «programa» o una «app» —abreviatura de application—.

Hay muchos otros procesos y sistemas funcionando en segundo plano para hacer posible todo esto. Sin embargo, no profundizaré en ellos en esta introducción. Si terminas de leerla y sientes curiosidad por aprender más, te recomiendo buscar en YouTube términos como «Binary», «Machine Code» y «BASIC Code».

¿Y qué importancia tiene esto? Podemos decirle a una computadora qué hacer, pero ¿por qué importa? En términos generales, una computadora, por definición, debe ser capaz de hacer estas cuatro cosas:

1. Recibir datos
2. Procesar datos
3. Almacenar datos
4. Producir datos

Estas cuatro capacidades nos permiten crear cosas a partir de nuestra propia imaginación. Puede que no sea obvio al principio, pero usar una computadora para crear cosas mediante código nos permite ser como un dios dentro de nuestro propio mundo virtual. Al menos, así es como yo lo veo...

## Datos

Espero no tener que explicar qué son los datos, pero puedes considerarlos como cualquier tipo de información. Aunque no tenga sentido para ti, cualquier tipo de información se considera un dato.

Por ejemplo, un libro de recetas está lleno de datos: contiene información sobre cómo preparar muchos tipos de comida. El libro puede estar escrito en inglés, español o incluso chino. A menos que sepas leer un idioma determinado, no podrás entender los datos escritos en ese idioma.

Lo mismo sucede con las computadoras. Las computadoras no entienden inglés, español ni chino. Entienden código binario: unos y ceros, «1» y «0». Como dije antes, no profundizaré en el sistema binario, pero es importante entender que no podemos comunicarnos con los chips de una computadora usando el mismo idioma que hablamos.

Para comunicarnos con las computadoras, inventamos nuevos lenguajes que fueran más fáciles de entender para nosotros y que una computadora pudiera traducir fácilmente a código binario. A esto lo llamamos «lenguaje de programación», y existen muchísimos lenguajes diferentes. De hecho, hay casi tantos lenguajes de programación como idiomas hablados.
